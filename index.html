Projects

Accurate Step Count with Deep Learning

Physical activity (PA) is globally recognized as a pillar of general health. Step count, as one measure of PA, is a well known predictor of long-term morbidity and mortality. Here I show that by using state-of-the-art neural network algorithms, I can achieve highly accurate step counts (98-99%) from diverse types of wearable and mobile devices. If you are interested, please find more details here.

Cancer Detection with Deep Learning

In this project, I tackle the CAMELYON16 challenge in which we have to detect cancer in hyper-resolution scans of human tissues (up to 100,000 x 100,000 pixels). To solve the problems, I experimented with several state-of-the-art neural network models including InceptionV3 and InceptionResnetV2). I show that we can obtain very high performance when using a pretrained InceptionV3 network and further tuning on a subset of the image. If you want to learn more, feel free to read the report and the source code here.

Understand Human Decision-making

This is essentially my PhD works in which I study how humans make decisions (esp. irrational ones) using a combination of computer-controlled behavioral experiments and machine learning (with the focus on Bayesian models). This work reveals a unifying principle behind many of human irrational decisions: we tend to keep consistency across our decisions so that any subsequent decisions are biased to be consistent with the preceding ones even when it is highly irrational. You can read the technical paper or non-technical article and play around with Matlab code.
